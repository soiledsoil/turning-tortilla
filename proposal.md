# turning-tortilla

## Repository
https://github.com/soiledsoil/turning-tortilla

## Description
It'll be a color picker that displays the hex and RGBA values of a part of the picture you choose.
I feel that it'll be helpful in keeping colors/styles consistent between creation of any media.

## Features
- Picture Importing and color picking
	- Allows you to drop/pick an image and choose a part of it to obtain it's color.
- Hex and RGBA value
	- Displays the Hex and RGBA values to be copied and used
- Color History
	- History of previously clicked colors

## Challenges
- Interative Ui.
- File importing/ File dropping.
- Hex Conversions.

## Outcomes
Ideal Outcome:
- The capability to import an image and choose the color of any pixel and output it's hex and RGBA values. With a short history of previous colors as well.

Minimal Viable Outcome:
- The capability of importing and picking the a color from the image with either the hex or RGBA values displaying.
